# Обсуждение RTLS-расширений (SRT 200–203)

**Дата:** 2026-06-11

## Описание
Расширения для Real-Time Locating System (Indoor-позиционирование) поверх стандартного EGTS.

### SRT 200 — RTLS Data
- BLE beacons, WiFi access points, NFC.
- RSSI, MAC, UUID.

### SRT 201–203
- Дополнительные метрики позиционирования.

## Применение в РНИС
- Indoor/Indoor-Outdoor трекинг на объектах (депо, склады Rolf, общественный транспорт).
- Комбинирование с GPS (POS_DATA).
